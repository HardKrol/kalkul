## Принцип работы

### Этот простенький калькулятор помогает выполнить следующие действия с двумя заданными числами:
   1) Сложение
   2) Вычитание
   3) Умножение
   4) Деление

## Внешний вид интерфейса
    Введите первое число: 1
    Введите второе число: 2

    Выберете математическую операцию:
    + : Сложение
    - : Вычитание
    / : Деление
    * : Умножение
